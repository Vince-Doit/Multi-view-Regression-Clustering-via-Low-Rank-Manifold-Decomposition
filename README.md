# Multi-view-Regression-Clustering-via-Low-Rank-Manifold-Decomposition

This repository contains the code for the paper titled "Multi-view Regression Clustering via Low-Rank Manifold Decomposition," which addresses the inherent conflict between scalability and geometry-consistent cluster assignment in similarity-graph-based multi-view clustering, as well as the anchor-number selection challenge in anchor-based extensions. The proposed method, MvMD, reformulates multi-view clustering as a unified regression problem that directly infers cluster labels without explicit graph construction or anchor selection, incorporating global balance regularization and low-rank manifold decomposition for robust and efficient clustering.

The repo also hosts some baseline systems as we compared in the paper. We would like to thank the authors of the baseline systems for their codes. If any baseline systems cannot be licensed freely here, please drop me an email, so we can remove it from the collection.

All experiments are conducted in MATLAB R2021b on a Linux operating system with an Intel Core i7-12700KF CPU and 64GB RAM.
